# Book Tracker

**Автор:** Имя Фамилия

## Описание
GUI-приложение для учёта прочитанных книг. Позволяет добавлять книги (название, автор, жанр, количество страниц), просматривать список, фильтровать по жанру и минимальному количеству страниц. Данные сохраняются в файл `books.json`.

1. Убедитесь, что установлен Python 3.7 или выше.
2. Скачайте файл `book_tracker.py`.

## Использование
- Заполните все поля и нажмите **«Добавить книгу»** — запись появится в таблице.
- Выберите жанр в выпадающем списке или введите минимальное количество страниц и нажмите **«Применить»** для фильтрации.
- Кнопка **«Сбросить»** отменяет фильтр и показывает все книги.
- Введённые данные автоматически сохраняются в `books.json`.

## Примеры тестирования
1. **Добавление:** введите «1984», «Оруэлл», «Антиутопия», «328» → нажмите Добавить. Книга отобразится в таблице.
2. **Валидация:** оставьте пустым поле «Автор» → программа покажет сообщение об ошибке.
3. **Фильтр:** добавьте несколько книг разных жанров; выберите жанр «Антиутопия» и нажмите Применить — таблица покажет только книги этого жанра.
4. **Сохранение:** закройте и снова откройте приложение — ранее добавленные книги останутся в таблице.
